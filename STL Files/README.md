# STL Files

This folder contains all the STL files for the various 3D printed parts for the static and dynamic solar panel systems

| File Name | Description | On Dynamic System? | On Static System? | Amount Used |
|---|---|---|---|---|
| AccelerometerMount.stl | Mount to affix the accelerometer to the back of the dynamic panel | ✓ | X | 1 |
| ActuatorControllerMount.stl | Mount to affix the linear actuator controller to the turntable | ✓ | X | 1 |
| CableHoleCap.stl | Cosmetic part to beautify the openings where wires go beneath the base | ✓ | X | 2 |
| CableSwivelHolder.stl | Holder to allow the cable connecting the turntable and the base to swivel | ✓ | X | 1 |
| DCLoadBracket.stl | Brackets to hold the DC loads to the base | ✓ | ✓ | 4 |
| EncoderGear.stl | Gear attached to encoder shaft that meshes with turntable gear | ✓ | X | 1 |
| EncoderHolder.stl | Holds the encoder on the platform | ✓ | X | 1 |
| HalfBBMount.stl | Mount to affix a half breadboard down | ✓ | X | 1 |
| JoystickHolder.stl | Holds the debug joystick on the paltform | ✓ | X | 1 |
| LDRMount.stl | Mount to affix a <u>l</u>ight <u>d</u>ependent <u>r</u>esistor to a panel's plywood backing | ✓ | ✓ | 8 |
| MotorGear.stl | Gear attached to servo shaft that meshes with turntable gear | ✓ | X | 1 |
| PanelClip.stl | Clip to hold a panel to it's backing | ✓ | ✓ | 6 |
| PanelClipTH.stl | Clip to hold a panel to it's backing and have a temperature/humidity sensor attached | ✓ | ✓ | 2 |
| PowerSupplyBracket.stl | Bracket to hold the power supply to the base | ✓ | X | 1 |
| ServoControllerHolder.stl | Holds the servo controller on the platform | ✓ | X | 1 |
| ServoMotorHolder.stl | Holds the servo motor on the platform | ✓ | X | 1 |
| StaticCableSupport.stl | Support for the cable between the arduino and panel to rest on | X | ✓ | 1 |
| StaticPanelHinge.stl | Hinge for a panel to articulate on if necessary | X | ✓ | 2 |
| StaticPanelHolder.stl | Holder for a panel to rest on | X | ✓ | 2 |
| TurntabeGearEighth.stl | Gear section attached to top of turntable | ✓ | X | 5 |
| UnoHalfBBMount.stl | Mount to affix an Arduino UNO and a half breadboard down | ✓ | ✓ | 2 |
